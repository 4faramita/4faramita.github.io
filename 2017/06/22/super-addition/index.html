1000. 函数求值

2017-06-22

Description

F(0, n) = n，对于所有的正整数n..
F(k, n) = F(k – 1, 1) + F(k – 1, 2) + … + F(k – 1, n)，对于所有的正整数k和n.

请实现下面Solution类中计算F(k, n)的函数（1 <= k, n <= 14）.

例1：F(1, 3) = 6

例2：F(2, 3) = 10

例3：F(10, 10) = 167960

Solution
