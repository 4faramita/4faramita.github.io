387. First Unique Character in a String

2017-03-07

Description

Difficulty: Easy

Given a string, find the first non-repeating character in it and return it’s index. If it doesn’t exist, return -1.

Example
s = “leetcode”
return 0.

s = “loveleetcode”,
return 2.

Note
You may assume the string contain only lowercase letters.

找到一个字符串中不重复的字符，返回其在字符串中位置，如不存在则返回 -1。

Solution

思路：维护一个字典，键为字符串中各个字符，值为这样一个数组：[该字符出现次数, 该字符首次出现位置]。
扫描一遍字符串，维护字典；
扫描一遍字典，维护出现次数为 1 的最小位置，返回这个值。
