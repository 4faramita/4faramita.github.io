171. Excel Sheet Column Number

2017-03-07

Description

Difficulty: Easy

Given a column title as appear in an Excel sheet, return its corresponding column number.

Example
A -> 1
B -> 2
C -> 3
…
Z -> 26
AA -> 27
AB -> 28

将 Excel 中使用字母表示的列号转换为数字。

Solution

26 进制转换为 10 进制。
